# DebtManagerApp2

**Система анализа и сегментации задолженностей по ЖКХ**

**Дата обновления:** 2 марта 2026 г.  
**Версия:** 1.1  
**Python:** 3.12+

---

## 📋 Описание

Приложение для обогащения данных ФССП и сегментации задолженностей по коммунальным платежам на основе данных биллинга (DBF файлы) и судебных реестров.

### Основные возможности

- **Загрузка данных** из Excel (ФССП, ЕИРЦ, УЖХ, ЕИАС) и DBF (биллинг)
- **Обогащение ФССП** — сопоставление исполнительных производств с судебными решениями
- **Сегментация долга** — расчёт непросуженного долга по каждому лицевому счёту
- **Экспорт результатов** в Excel с шаблонами
- **GUI интерфейс** на CustomTkinter с прогресс-баром и журналами

## 📊 Как это работает

### Часть 1: Обогащение ФССП

1. Загружается файл ФССП с исполнительными производствами
2. Загружаются судебные реестры (ЕИРЦ, УЖХ, ЕИАС)
3. Загружаются DBF файлы биллинга
4. Производится сопоставление по номеру дела или ФИО+адресу
5. Рассчитываются суммы (ОД + пени + госпошлина)
6. Проверяется равенство с суммой из ФССП

### Часть 2: Сегментация

1. Для каждого ЛС определяется дата отсечки (максимальная `date_end` из судебных решений)
2. Рассчитывается глубина задолженности (количество месяцев до даты отсечки)
3. Определяется сумма непросуженного долга
4. Результаты сохраняются в таблицу `processed_segmentation`

## 📈 Результаты

После обработки в папке `output/` создаются:
